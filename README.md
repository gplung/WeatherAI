
# Multi-City Weather Dashboard

Features:
- Up to 5 cities
- 10-day forecasts
- Synchronized horizontal scrolling
- Rain chance
- Wind at hottest part of day
- Humidity at hottest part of day
- Mobile-first iPhone layout
- Uses free Open-Meteo APIs

## Deploy to GitHub Pages

1. Create a new GitHub repository
2. Upload all files
3. Go to:
   Settings → Pages
4. Source:
   Deploy from branch
5. Choose:
   main / root
6. Save

Your app will appear at:
https://YOURNAME.github.io/REPO-NAME/
